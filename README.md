# Crypto Java Project

## 📌 Description
Ce projet contient deux applications :

### 1. Application Console
- Chiffrement AES
- Déchiffrement
- Hash SHA-256
- Comparaison de hash

### 2. Application Web (Spring Boot)
- API REST (/crypt, /decrypt, /hash)
- Interface web HTML
- Ouverture automatique du navigateur

## 🚀 Technologies
- Java
- Spring Boot
- HTML / CSS / JavaScript

## ▶️ Lancer le projet

### Web App
mvn spring-boot:run

Puis ouvrir :
http://localhost:8080
